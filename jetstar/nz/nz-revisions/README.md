# Jetstar NZ Revision Documents

This folder contains revision documents for the Jetstar NZ email templates.

## File Mapping

| Revision Document | Email Template |
|---|---|
| `cancel-jetstar-nz.docx` | [cancel.html](../en/cancel.html) |
| `policy-confirmation-nz.docx` | [policy-confirmation.html](../en/policy-confirmation.html) |
| `post-trip-jetstar-nz.docx` | [post-trip.html](../en/post-trip.html) |
| `pre-trip-jetstar-nz.docx` | [pre-trip.html](../en/pre-trip.html) |

## Naming Convention

- Revision documents follow the pattern: `[email-type]-jetstar-nz.docx`
- The corresponding email template is located at: `../en/[email-type].html`
